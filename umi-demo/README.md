# umi project

## 创建项目
 
 ```bash
 yarn create @umijs/umi-app
```

创建下来是`.tsx`后缀的文件，可将`.tsx`后缀改成`.jsx`后缀

## 开始项目

下载依赖

```bash
 yarn
```

运行

```bash
 yarn start
```
